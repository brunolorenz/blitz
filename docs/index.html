Carregando dados…
Dados via API pública do Chess.com (leaderboards + arquivos de partidas), atualizados periodicamente por uma GitHub Action. Horários de encerramento de partida. "Estimado por país" e "Horário de Brasília" são conversões aproximadas (fuso da capital do país, sem ajuste de horário de verão — exceto o Brasil, que não observa mais horário de verão desde 2019, então esse offset é exato). Aberturas são derivadas do slug de abertura do Chess.com por uma heurística; nomes podem ocasionalmente vir truncados.
