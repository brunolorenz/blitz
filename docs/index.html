O relógio dos top blitz

Quando os melhores jogadores de blitz do mundo realmente jogam — hora a hora, dia a dia, no horário de Brasília.

Carregando dados…
Dados via API pública do Chess.com (leaderboards + arquivos de partidas), atualizados periodicamente por uma GitHub Action. Horários convertidos para Horário de Brasília (UTC-3, fixo — o Brasil não observa mais horário de verão desde 2019, então essa conversão é exata). Aberturas são derivadas do slug de abertura do Chess.com por uma heurística; nomes podem ocasionalmente vir truncados. As "melhores janelas" e percentuais são baseados no histórico de partidas do período, não em presença ao vivo.
