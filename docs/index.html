O relógio dos top blitz

Quando os melhores jogadores de blitz do mundo realmente jogam — e quando LPSupi tem mais chance de encontrar cada um deles, em horário de Brasília.

Carregando dados…
Como esses dados são construídos (leia antes de confiar demais nos números)
  • Só partidas blitz 3+0 (3 minutos, sem incremento) contam. Quem joga mais 3+2 ou 5+0 aparece com bem menos partidas aqui do que joga na vida real.
  • Os dados cobrem só o histórico recente (1 mês para o ranking geral, 3 meses para o LPSupi) — não é a carreira toda, é uma amostra.
  • Todo horário já vem convertido para horário de Brasília (UTC-3, fixo, sem horário de verão).
  • "Sobreposição": para cada jogador, é a % das partidas dele que caíram em algum bloco de dia+hora onde o LPSupi já jogou pelo menos uma vez no período. Não é probabilidade de encontro ao vivo — é só um indício baseado em histórico.
  • O explorador de horário usa a mesma lógica, mas para qualquer dia/hora que você escolher, olhando só para a rotina de cada jogador (não a do LPSupi).
  • Aberturas são adivinhadas a partir da URL que o Chess.com associa a cada partida — é uma heurística de texto, então o nome pode vir cortado ou impreciso em alguns casos.
  • Amostras pequenas (poucas partidas) deixam qualquer % pouco confiável — desconfie de picos com poucas partidas de base.
Dados via API pública do Chess.com (leaderboards + arquivos de partidas, blitz 3+0), atualizados periodicamente por uma GitHub Action. Horários convertidos para Horário de Brasília (UTC-3, fixo). Percentuais de "sobreposição" e do explorador são baseados no histórico de partidas do período, não em presença ao vivo. Aberturas são derivadas do slug de abertura do Chess.com por uma heurística; nomes podem ocasionalmente vir truncados.
